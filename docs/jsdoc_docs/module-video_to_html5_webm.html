Module: video_to_html5_webm

Converts audio or video file into a HTML5 video webm.
uses code from this issue/question from fluent-ffmepg
to understand the ffmpeg used to encode webm video see this guide

Path to the ffmpeg binary. Optional, if no provided it will try to use system one if present.
