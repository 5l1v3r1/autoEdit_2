Properties
Name Type Description
title string Title of the EDL video sequence
offset string The camera timecode, eg free run, rec run, time of day, repsent a time offset from the time relative to the beginning of the video file timeline. in format of timecode eg "00:00:28:08" such as "hh:mm:ss:ms" because that's how cameras write it in the metadata of the file.
events Array.<Object> array of video segment makind up the EDL sequence
Properties
Name Type Description
id string "id" of video segment
startTime number start time of video segment in seconds
endTime number end time of video segment in seconds
reelName string reel name of video segment, generally the name of the card the footage was filmed on or not available
clipName string file name that the video segment belongs to. Only file name. no path.
