Module: split

Takes in an audio file, if it exceeds 5 minutes, it splits at 5 minutes intervalls.
Used by transcriber/index.js module.

(inner) split(file, tmpFolder, ffmpegBinPath, ffprobeBinPath, callback) → {callback}

splits an audio file, if it exceeds 5 minutes, in 5 minutes intervalls.
using ffprobe to read duration. ffmpeg passed to trimmer module.
saves trimmed clips in temp folder.

Parameters:
Name Type Description
file string

audio file path
tmpFolder string

path to a temporary folder to put temporary audio files created when dividing audio file into chunks to send to STT API.
ffmpegBinPath string

path to ffmpeg binary. to pass to trimmer module
ffprobeBinPath string

path to ffprobe binary. If not present it will try to use system one.
callback callback
